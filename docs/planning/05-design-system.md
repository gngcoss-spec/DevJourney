# Design System - DevJourney

## 1. 디자인 원칙

### 1.1 핵심 가치
1. **명확함 (Clarity)** - 정보가 한눈에 파악되어야 함
2. **최소주의 (Minimalism)** - 불필요한 장식 제거, 내용에 집중
3. **일관성 (Consistency)** - 같은 패턴은 같은 방식으로
4. **접근성 (Accessibility)** - 컬러만으로 구분하지 않음

### 1.2 디자인 무드
- **Jira의 기능성** + **Linear의 깔끔함** + **Notion의 편안함**
- 전문적이지만 부담스럽지 않은 느낌
- 다크 모드 우선 (개발자 도구 특성)

## 5. 컴포넌트 가이드

### 5.1 카드 (Service Card, Work Item Card)

```
┌─────────────────────────────────┐
│  [상태 뱃지]     [진행률 38%]   │
│                                 │
│  서비스명                       │
│  현재 단계: 개발                │
│                                 │
│  마지막 활동: 2일 전            │
│  다음 액션: API 인증 구현       │
└─────────────────────────────────┘
```

- 배경: Surface (Slate 900)
- 테두리: Border (Slate 700) 1px
- 모서리: rounded-lg (8px)
- 호버: Surface Elevated (Slate 800) + 테두리 색 밝아짐
- 패딩: 16px

### 5.2 Kanban 컬럼

```
┌─── Backlog ────┐
│ [+ 새 항목]    │
│                │
│ ┌────────────┐ │
│ │ Work Item  │ │
│ │ Card       │ │
│ └────────────┘ │
│ ┌────────────┐ │
│ │ Work Item  │ │
│ │ Card       │ │
│ └────────────┘ │
└────────────────┘
```

- 컬럼 배경: 투명
- 컬럼 헤더: 상태 색상 도트 + 텍스트 + 카운트
- 카드 간격: 8px

### 5.3 모달

- 너비: 640px (Work Item 상세)
- 배경: Surface Elevated (Slate 800)
- 오버레이: 검정 50% 불투명
- 모서리: rounded-xl (12px)
- 탭 네비게이션 상단

### 5.4 사이드바

- 너비: 240px (데스크톱), 접힘 가능
- 배경: Surface (Slate 900)
- 활성 메뉴: Blue 500 배경 + 흰 텍스트
- 호버: Slate 800 배경
